Welcome to Scriptless Pipes!

This entire game was made using only HTML and CSS!

The goal of this game is to get to the bottom of the webpage!

You do this by clicking on the pipes! Each time you click on a pipe, you will be transported to the other end of the pipe.

Each start and end of a pipe exists on a "layer".

You want to visit layers via the pipes and slowly figure out where each goes and what pipes you need to get to the bottom.

This game is playable on mobile, but I recommend playing in landscape mode, or else things might get too bunched together
